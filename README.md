# 家庭記帳簿


## 功能介紹
#### 須登入後才能使用
#### 可以facebook帳號註冊
#### 使用者可自行新增一筆支出
#### 使用者可自行刪除任一筆支出
#### 使用者可自行修改任一筆支出
#### 使用者可選取支出類別並顯示總金額
#### ※使用完畢請記得登出


## 使用方式
1. 請事先安裝好Node.js與Express.js
2. clone本專案至本地
3. 從終端機進行專案放置路徑
4. 輸入`npm run start`以啟動專頁
5. 若看到`Succeed in running.`表示啟動成功
